# Cronos Creations Portfolio

A modern, interactive portfolio website for a graphic designer and typography specialist.
Built with HTML, CSS, and Vanilla JavaScript.

## Features
- **Dynamic Boot Animation**: A custom "Design App" loader sequence.
- **Glassmorphism UI**: Premium frosted glass effects on menus and cards.
- **Interactive Elements**: Custom cursor, magnetic buttons, and 3D particle background.
- **Responsive Design**: Fully optimized for mobile and desktop.

## Hosting on GitHub Pages
1. Go to your repository **Settings**.
2. Navigate to **Pages**.
3. Select the **main** branch as the source.
4. Click **Save**.
The site will be live at `https://[username].github.io/[repo-name]/`.
